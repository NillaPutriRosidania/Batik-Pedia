## About our Model

The model we use uses a CNN architecture which is useful for finding patterns in images to recognize objects, classes and categories.  
Where we use 10 classes: 
- Batik Betawi Batik
- Batik Blekok
- Batik Cendrawasih 
- Batik Geblek Renteng 
- Kawung Batik
- Batik Lasem 
- Batik Megamendung
- Batik Parang
- Batik Poleng
- Batik Sekar Jagad
